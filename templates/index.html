How PR Shield Works

Three specialized AI agents collaborate through the Band SDK to review your code from every angle.

🔍
1. Analyst Agent — AI/ML API
Reads the PR diff and understands what the developer was actually trying to do. Maps code changes to intent. Flags complexity and areas that deserve extra scrutiny.
🔒
2. Security Agent — Featherless AI
Scans for SQL injection, XSS, hardcoded secrets, auth flaws, and insecure patterns. Runs a static regex scan first, then deep AI analysis. Runs in parallel with the Analyst.
🧑‍💻
3. Mentor Agent — Google AI/ML API
Synthesizes both agents' findings into a complete code review. With Style Memory, it writes in the voice of YOUR senior developer — not generic AI feedback.
🧠
Style Memory Engine — The Twist
Feed it merged PRs from your history. It extracts your team's coding DNA: naming conventions, test expectations, what your senior dev pushes back on. Each review gets more personalized over time.
